# Cisco Packet Tracer Lab 1 - Device Connections and Cable Selection

## Overview

This lab demonstrates the fundamentals of connecting networking devices in Cisco Packet Tracer using the appropriate cable types. The objective is to understand how different network devices communicate and to identify the correct physical media required for each connection.

## Objectives

- Connect common networking devices in Cisco Packet Tracer.
- Select the correct cable type for each connection.
- Understand the purpose of different network cables.
- Verify that all physical connections are correctly established.

## Devices Used

- PCs
- Switches
- Routers
- Servers

## Cable Types Used

| Cable | Used For |
|--------|-----------|
| Copper Straight-Through | PC ↔ Switch, Switch ↔ Router |
| Copper Cross-Over | Switch ↔ Switch, Router ↔ Router, PC ↔ PC |
| Console Cable | PC ↔ Router/Switch Console Port |
| Fiber Optic  | High-speed switch connections |

## Skills Demonstrated

- Network topology creation
- Physical network design
- Cable selection
- Basic Cisco Packet Tracer navigation
